# Annual Vendor Review Process

## Purpose

This document defines the annual vendor re-review process for all in-scope vendors.

## Review Activities

Each annual review must include:

1. Collection of updated SOC reports
2. Review for material changes
3. Update of vendor risk assessments
4. Documentation of review findings

## Review Frequency

Vendor reviews are conducted annually.

## Responsibilities

- Security Team: Conduct review
- Compliance Team: Track completion
- Vendor Owner: Provide required documentation

## Required Evidence

- Latest SOC 2 / SOC 1 report
- Security questionnaires (if applicable)
- Risk assessment updates
- Review notes and approvals

## Documentation

Use `template.md` for all annual vendor reviews.

## Follow-up Actions

Any identified risks or findings must be tracked according to vulnerability management procedures.
